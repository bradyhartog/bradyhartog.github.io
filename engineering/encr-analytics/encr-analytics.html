ENCR Analytics

Reaction Engineering International

August 2020

Background

The Corrosion Probe Dashboard is a webapp that provides a wireless interface to an electronic probe for measuring electrochemical noise corrosion rate (“ENCR”) in industrial combustion furnaces. The Dashboard displays the readings of the probe in real time.

Problem

A raw ENCR signal has a low signal-to-noise ratio. This requires that the signal be filtered before any meaningful inferences can be drawn. The Corrosion Probe Dashboard lacks an integrated tool for implementing this essential processing step.

Objective

Design and implement an integrated tool for filtering the ENCR signal on the Corrosion Probe Dashboard.

Process

The filtering action is implemented with Python scripting. The Dashboard's API was expanded to facilitate transferring the filtered data from the scripts in the backend to the graphs in the frontend.

Outcome

The resulting tool, called ENCR Analytics, allows the user to interactively filter the ENCR signal using either a classic lowpass filter or a dedicated trend analyzer.
